When Outsourcing unsuitable features or partners (R93) occurs, it can cascade into Uncontrolled handling of migration changes (R90) because A Request For Change appears during migration.

# Reasoning Notes
Outsourcing unsuitable features or partners (R93) produces the consequence(s) 'Misalignment with strategic objectives, reduced control, inappropriate allocation of migration work.', which aligns with the root cause of Uncontrolled handling of migration changes (R90): The migration process contains many dependencies and changing business/architecture priorities..
