# Secondary Risk Created
Adds governance structure and coordination overhead.

# Applicability Conditions
Large-scale, long-term legacy-to-SOA migration.

# Observed Outcome
A central Program Management Committee coordinates technical, organizational, and business aspects of migration and provides governance for a large-scale SOA modernization.

# Reasoning Notes
Program Management Committee governance (M82) targets the root cause of Uncontrolled handling of migration changes (R90): The migration process contains many dependencies and changing business/architecture priorities.

# Standardized Relationship Entry
IF Create a central governing body with steering, core, program management, business change management, and architecture teams to manage the migration and align business, organizational, and technical decisions THEN the risk of uncontrolled handling of migration changes (R90) is prevented.
