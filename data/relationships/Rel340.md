# Applicability Conditions
Migration from highly optimized mainframe environments to distributed systems

# Observed Outcome
When Post-Modernization Performance Degradation (R10) occurs, it can cascade into User Resistance (R9) because Deployment of the new graphical user interface.

# Reasoning Notes
Post-Modernization Performance Degradation (R10) produces the consequence(s) 'Slower response times; failure to meet original performance expectations.', which aligns with the root cause of User Resistance (R9): Users are optimized for specific workflows and do not want to change.

# Standardized Relationship Entry
IF Post-Modernization Performance Degradation (R10) materializes THEN it can trigger or worsen User Resistance (R9).
