# Description

Develop a comprehensive strategy for data migration at the very beginning of the project, including mapping and cleaning.

# Prerequisites

Deep understanding of the legacy data model

# Tools / Technologies



# Trade-offs / Side Effects

May increase initial project duration due to extensive early planning requirements.

# Effectiveness

Medium

# Stakeholders

- Data Architect
- Project Manager

# Contextual Constraints

- Have to be done before migrating data


# Standardized Mitigation Entry

Developing a comprehensive data migration strategy early in the modernization lifecycle is essential to mitigate the risk of data loss and system failure.

# Key Quote for Prompting

> "If you are doing migration of your legacy, you have to prepare a good data migration strategy"

# Logic Pattern

IF [Early Strategic Planning] THEN [Reduced Risk of Migration Disturbance]
