# Description
Industry should collaborate closely with federal agencies on modernization plans and execution, bringing new ways to transition older systems and software to current industry-prominent hardware and software platforms.

# Prerequisites
Agency-industry collaboration model; industry modernization expertise; modernization plans ready for execution.

# Tools / Technologies
Current industry-prominent hardware and software platforms
