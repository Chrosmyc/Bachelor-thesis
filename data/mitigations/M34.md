# Description
Use a structured evaluation framework for cloud/hybrid-cloud platform selection based on resilience, regulatory compliance, and operational fit.

# Prerequisites
Cloud migration goals and compliance/resilience requirements.

# Tools / Technologies
Hybrid cloud selection framework

# Trade-offs / Side Effects
Evaluation period averages 4.6 months.

# Effectiveness
Following structured selection framework achieves 43% higher success rates.

# Stakeholders
- Architects; compliance teams; executives

# Contextual Constraints

- Financial institutions selecting cloud platforms.


# Standardized Mitigation Entry
Structured cloud selection reduces platform-fit risk by evaluating resilience, compliance, and architecture suitability before migration.

# Logic Pattern
