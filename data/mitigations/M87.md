# Description
Outsource only low-business-value features when outsourcing supports strategic objectives, follows outsourcing guidelines, uses strategic partners, and no other option is viable.

# Prerequisites
Outsourcing strategy guidelines; strategic partner availability; confirmation that no other option is viable.

# Tools / Technologies
Outsourcing strategy-guidelines

# Trade-offs / Side Effects
Reduced direct control over outsourced feature development.

# Effectiveness
The paper reports outsourcing helps the organization focus on high-priority features.

# Stakeholders
- Lead architects; strategic partners; program management committee

# Contextual Constraints

- Low-priority features such as card authorization and card payments.


# Standardized Mitigation Entry
Controlled outsourcing mitigates focus and cost risks by limiting external development to low-priority features under strict strategic guidelines.

# Key Quote for Prompting
> "This helps us to focus on the high priority features."

# Logic Pattern
IF a feature is low priority and outsourcing criteria are fulfilled THEN APPLY controlled outsourcing.
