# Description
Validate that new skills required by language replacement, cloud implementation, or changed hosting are in place; address role/responsibility changes in advance to facilitate adoption.

# Prerequisites
Target technology and operating model known.

# Tools / Technologies
Skills assessment; training/upskilling plan; role definitions.

# Trade-offs / Side Effects
Requires organizational change effort before full adoption.

# Effectiveness
Intended to facilitate cloud adoption and reduce disruption.

# Stakeholders
- IT personnel; operations; management; support contractors

# Contextual Constraints

- New core technology language
- public/private cloud
- changed hosting model.


# Standardized Mitigation Entry
Assess and prepare skills, roles, and responsibilities before adopting new technology or cloud hosting.

# Key Quote for Prompting
> "Address this risk in advance"

# Logic Pattern
IF modernization changes technology or hosting roles THEN APPLY skills validation and advance organizational preparation.
