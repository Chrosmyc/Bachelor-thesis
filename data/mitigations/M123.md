# Description
Use AI to reduce or eliminate manual tasks in the modernization process.

# Prerequisites
Manual modernization tasks can be automated or supported by AI

# Tools / Technologies
AI process automation

# Trade-offs / Side Effects


# Effectiveness
Reducing or eliminating manual tasks is reported as an AI use case by 51% of organizations using or planning AI for modernization.

# Stakeholders
- Developers
- architects
- operations teams
- project teams

# Contextual Constraints

- Applicable where repetitive or manual tasks exist


# Standardized Mitigation Entry
AI-supported automation reduces manual effort and may shorten modernization activities.

# Key Quote for Prompting
> "Reducing or eliminating manual tasks"

# Logic Pattern
IF modernization work contains repetitive manual tasks THEN use AI to reduce or eliminate them.
