# Description
Embeds automated and manual checkpoints with explicit acceptance criteria before data advances to the next pipeline stage.

# Prerequisites
Defined data quality acceptance criteria and pipeline checkpoints.

# Tools / Technologies


# Trade-offs / Side Effects


# Effectiveness
The paper reports 99.9% data reliability and elimination of persistent data quality issues in a Fortune 500 industrial manufacturer engagement.

# Stakeholders
- Data engineers
- analytics teams
- platform teams

# Contextual Constraints

- Applies to ingestion
- transformation
- and downstream consumption stages in modernized data platforms.


# Standardized Mitigation Entry
Data quality gates mitigate data reliability risk by validating data at defined pipeline stages before downstream advancement. In the paper, this structure is linked to 99.9% data reliability in a modernization case.

# Key Quote for Prompting
> "Data quality gates — automated and manual checkpoints embedded in the pipeline at defined intervals, with explicit acceptance criteria before data advances to the next stage."

# Logic Pattern
IF data advances through migration pipelines THEN APPLY data quality gates with acceptance criteria.
