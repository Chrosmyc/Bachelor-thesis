# Description
Use manual search across relevant software engineering sources and supplement it with snowballing to reduce the chance of missing relevant modernization studies due to inconsistent terminology.

# Prerequisites
Defined research protocol and selected research sources.

# Tools / Technologies


# Trade-offs / Side Effects
Manual searching requires additional effort and may still involve selection bias.

# Effectiveness
The authors state this strategy was adopted because modernization terms are not well defined and relevant articles might be ignored by search strings.

# Stakeholders
- Researchers

# Contextual Constraints

- Applicable to literature-based extraction and consolidation of modernization knowledge.


# Standardized Mitigation Entry
Manual search combined with snowballing helps capture modernization literature despite inconsistent terminology.

# Key Quote for Prompting
> "this manual strategy would allow us to find relevant articles that might be ignored in the case we used an approach based on search strings in digital libraries"

# Logic Pattern
IF modernization terminology is not well defined THEN APPLY manual search supplemented by snowballing.
