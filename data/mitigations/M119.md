# Description
Evaluate all associated IT and operational costs together, including platform, software, maintenance, wages, datacenter expenses, downstream costs, new hardware/software, and retraining.

# Prerequisites
Access to cost data across current and proposed environments

# Tools / Technologies
Cost metrics

# Trade-offs / Side Effects


# Effectiveness
Presented as important for modernization progress measurement and decision-making.

# Stakeholders
- Budget owners; program leadership; procurement; operations teams; modernization team

# Contextual Constraints

- Applies when comparing current systems with new-system alternatives or when tracking cost savings.


# Standardized Mitigation Entry
Full cost metrics reduce modernization cost-evaluation risk by capturing current, downstream, new-system, and retraining costs in a comparable decision basis.

# Key Quote for Prompting
> Paraphrased evidence: all associated costs should be evaluated together.

# Logic Pattern
IF modernization decisions rely on cost comparison THEN APPLY full cost metrics across current and target environments.
