# Description
Conduct deep upfront planning, systematic evaluation of trade-offs, and identification of risks before project start.

# Prerequisites
Clear business goals, stakeholder alignment.

# Tools / Technologies
Risk analysis frameworks, evaluation methodologies.

# Trade-offs / Side Effects
May delay actual project execution start.

# Effectiveness
High (noted as the most important factor for success).

# Stakeholders
- Project Manager
- Software Architect

# Contextual Constraints

- Essential for large-scale projects.


# Standardized Mitigation Entry
Systematic upfront planning and risk evaluation are critical to avoid project failure

# Key Quote for Prompting
> "The most important factor to success is to carefully plan large migration projects and consider all the options..."

# Logic Pattern
IF [Systematic Planning] THEN [Reduced Failure Risk]
