# Description
Operate the legacy system in parallel as a hot backup during modernization deployment.

# Prerequisites
Ability to synchronize databases.

# Tools / Technologies
Parallel Operations

# Trade-offs / Side Effects
Strongly increases maintenance and synchronization logic complexity.

# Effectiveness
Reduces operational deployment risk.

# Stakeholders
- DevOps Engineer
- Project Manager

# Contextual Constraints

- Applies to high-availability systems.


# Standardized Mitigation Entry
Parallel operation of legacy systems provides fallback capability during modernization deployment.

# Key Quote for Prompting
> "The legacy system is maintained as a 'hot' backup."

# Logic Pattern
IF legacy and modernized systems run in parallel THEN operational fallback capability exists.
