# Description
Use expert guidance throughout the migration journey and define a prescriptive path for overcoming migration hurdles.

# Prerequisites
Cloud migration or modernization initiative; awareness that migration challenges may occur

# Tools / Technologies
AWS migration and modernization services; AWS Migration Competency Partner guidance

# Trade-offs / Side Effects


# Effectiveness
The paper states that expert guidance helps organizations anticipate challenges and accelerate migration benefits.

# Stakeholders
- C-suite leaders
- IT leaders
- in-house IT teams
- AWS Partner / IBM Consulting

# Contextual Constraints

- AWS cloud migration and modernization of enterprise workloads

Expert guidance and a prescriptive migration path help reduce the chance that modernization is slowed by unanticipated migration hurdles.
