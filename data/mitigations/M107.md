# Description
Defines which engineering workflows are eligible for AI-assisted automation, which require human review, and which are excluded from GenAI approaches.

# Prerequisites
Defined modernization workflows and governance boundaries for AI use.

# Tools / Technologies
GenAI tools / AI-assisted automation

# Standardized Mitigation Entry
Use-case scoped guardrails reduce AI-output risk by defining where AI assistance is allowed, where review is mandatory, and where GenAI is excluded. The guardrails specify governance conditions for safe and auditable AI acceleration.

# Key Quote for Prompting
> "Use-case scoped guardrails — explicit boundaries defining which engineering workflows are eligible for AI-assisted automation."

# Logic Pattern
IF GenAI is used in engineering workflows THEN APPLY use-case scoped guardrails.
