# Description
Ensure modernization efforts comply with the organization's SDLC; if the current SDLC does not map to modernization, modify and standardize it across modernization programs.

# Prerequisites
Existing or adaptable SDLC process.

# Tools / Technologies
SDLC process; modernization standards.

# Trade-offs / Side Effects
May require process updates before execution.

# Effectiveness
Provides established rigor for modernization execution.

# Stakeholders
- Program manager; development team; quality assurance; governance board

# Contextual Constraints

- Execution phase.


# Standardized Mitigation Entry
Enforce or adapt SDLC discipline so modernization follows standardized development and maintenance rigor.

# Key Quote for Prompting
> "subject to/enforced by the established rigor"

# Logic Pattern
IF modernization execution begins THEN APPLY SDLC compliance or standardization.
