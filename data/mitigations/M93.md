# Description
Base roadmap cost and time estimates on a genuine assessment of the existing application rather than guesses.

# Prerequisites
Completed application assessment and baseline.

# Tools / Technologies
Cost estimation; roadmap; baseline metrics.

# Trade-offs / Side Effects
Requires additional assessment time before funding decisions.

# Effectiveness
The paper calls this a major reason for assessment as the first step.

# Stakeholders
- Finance; program manager; senior leadership

# Contextual Constraints

- Business case and roadmap development.


# Standardized Mitigation Entry
Ground modernization estimates in a current-state assessment to reduce unrealistic cost and schedule assumptions.

# Key Quote for Prompting
> "always conducting an assessment as the first step"

# Logic Pattern
IF cost and schedule estimates are needed THEN APPLY genuine assessment-based estimation.
