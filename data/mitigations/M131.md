# Description
Use tools such as Azure Migrate and other assessment products to automate discovery of workload components, configurations, compatibility issues, databases, and applications.

# Prerequisites
Access to workload environments and supported discovery/assessment tooling.

# Tools / Technologies
Azure Migrate; Dr Migrate; Cloudockit; CAST Highlight; CloudPilot; GitHub Copilot app modernization

# Trade-offs / Side Effects
Tools reduce manual effort but might miss undocumented dependencies.

# Effectiveness
Provides consistent data collection and systematic analysis across the environment.

# Stakeholders
- Migration team
- architects
- workload owners

# Contextual Constraints

- Applies during assessment before moving workloads to Microsoft Azure.


# Standardized Mitigation Entry
Automated discovery and assessment tools provide consistent workload visibility and help detect configuration, database, and compatibility issues before Azure migration.

# Key Quote for Prompting
> "automate discovery of workload components and configurations"

# Logic Pattern
IF workload visibility is needed THEN APPLY automated discovery and assessment tools.
