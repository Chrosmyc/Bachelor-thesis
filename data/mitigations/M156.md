# Description
Estimate the cost of customizing ERP or application systems to integrate required cloud or on-premises applications for a new business model.

# Prerequisites
Target business model, required integrations, current ERP/API capabilities, and customization cost assumptions.

# Tools / Technologies
ERP integration analysis; API assessment; cloud/on-prem application integration assessment

# Trade-offs / Side Effects


# Effectiveness
The paper recommends asking this cost question in the CEO/CFO business case; no quantified effectiveness is reported.

# Stakeholders
- CIO; CFO; ERP team; application architects; business model owners

# Contextual Constraints

- Applies when modernization must support a new business model through cloud or on-premises application integration.


# Standardized Mitigation Entry
Estimate customization and integration costs early to reduce budget uncertainty and delivery risk for modernization-enabled business models.

# Key Quote for Prompting
> [paraphrased] Ask what it costs to customize ERP for required new application integration.

# Logic Pattern
IF a new business model requires cloud/on-prem integration THEN APPLY customization and integration cost estimation.
