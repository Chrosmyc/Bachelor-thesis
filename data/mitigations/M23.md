# Description
Use wrappers, shells, and adapters to preserve legacy interfaces while migrating functionality.

# Prerequisites
Interface specifications must be available.

# Tools / Technologies
Adapters, Wrappers

# Trade-offs / Side Effects
Adds architectural complexity by maintaining intermediary layers.

# Effectiveness
Supports incremental modernization compatibility.

# Stakeholders
- Software Architect
- Developer

# Contextual Constraints

- Applies during application-based migration.


# Standardized Mitigation Entry
Wrappers and adapters can preserve compatibility between legacy and modernized components during incremental migration.

# Key Quote for Prompting
> "It is necessary to develop a shell and adapter."

# Logic Pattern
IF wrappers and adapters preserve legacy interfaces THEN incremental compatibility improves.
