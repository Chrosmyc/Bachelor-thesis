# Description
Analyze conflicts among driving forces, needs, constraints, resources, and expectations with relevant stakeholders.

# Prerequisites
Multiple modernization goals or constraints exist.

# Tools / Technologies


# Trade-offs / Side Effects
Can slow the decision phase but reduces goal conflict.

# Effectiveness
The paper recommends trade-off analysis when conflicts emerge among driving forces.

# Stakeholders
- Managers; engineers; developers; customers

# Contextual Constraints

- Applies when modernization drivers are diverse or conflicting.


# Standardized Mitigation Entry
Use stakeholder trade-off analysis to make conflicting modernization drivers explicit and reduce misaligned modernization decisions.

# Key Quote for Prompting
> As conflicts can emerge among those driving forces, a trade-off analysis could be performed by the stakeholders.

# Logic Pattern
IF driving forces conflict THEN APPLY stakeholder trade-off analysis.
