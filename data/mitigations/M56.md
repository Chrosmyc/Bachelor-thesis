# Description
Identify microservice candidates by investigating coupling and cohesion among legacy parts, considering migration of whole features, and focusing on reused parts of the legacy system.

# Prerequisites
Existing features; dependencies between features; data dependency; data flow; list of technologies used in the legacy system.
