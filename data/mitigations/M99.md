# Description
Use discovery tools and cognitive agents to gain a clear view into legacy applications and determine which modernization approach delivers the most value for each targeted application.

# Prerequisites
Access to application landscape and assessment tooling.

# Tools / Technologies
Discovery tools; cognitive agents.

# Trade-offs / Side Effects
Requires tooling and analysis capability.

# Effectiveness
Provides clear view into applications for selecting value-maximizing approaches.

# Stakeholders
- Architecture staff; systems integrator; IT; business owners

# Contextual Constraints

- Digitization and application selection.


# Standardized Mitigation Entry
Use discovery tools and cognitive agents to understand applications and select modernization approaches based on value and impact.
