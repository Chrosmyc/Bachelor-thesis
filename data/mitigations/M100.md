# Description
Engage with industry and partner with vendors to support complex modernization, integration, hosting, telecommunications, and solution development.

# Prerequisites
Vendor/industry stakeholders identified and governed.

# Tools / Technologies
Vendor collaboration; industry outreach.

# Trade-offs / Side Effects
Requires oversight and management of external partners.

# Effectiveness
Multiple case examples list vendor/industry partnership as a key success factor.

# Stakeholders
- Vendors; industry partners; government project office; contracting; IT

# Contextual Constraints

- Complex IT modernization involving commercial providers or external stakeholders.


# Standardized Mitigation Entry
Use managed vendor and industry partnerships to reduce third-party and implementation risks in complex modernization.

# Key Quote for Prompting
> "government and vendor collaboration"

# Logic Pattern
IF modernization relies on external providers THEN APPLY managed vendor and industry partnership.
