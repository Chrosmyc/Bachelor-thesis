# Description
Start modernization by strengthening data foundations, streamlining legacy systems, focusing on security, and aligning IT initiatives with business objectives and customer outcomes.

# Prerequisites
Assessment of existing complexity; data foundation work; business objective alignment.

# Tools / Technologies


# Trade-offs / Side Effects
May delay visible deployment of new tools.

# Effectiveness
The paper presents simplification, security, and strategic alignment as the way to unlock transformation without outdated-technology traps.

# Stakeholders
- CIO; enterprise architects; data teams; security teams; business leaders

# Contextual Constraints

- Applies before deploying new technologies such as AI on complex existing foundations.


# Standardized Mitigation Entry
Simplifying architecture and strengthening data foundations before deploying new tools reduces bolt-on modernization, scalability, integration, and security risks.

# Key Quote for Prompting
> 

# Logic Pattern
IF new tools are planned on complex foundations THEN APPLY simplification and data foundation strengthening first.
