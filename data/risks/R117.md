# Description
SAP workloads are mission-critical; selecting a modernization path or platform that does not fit SAP workload requirements may fail to deliver cost, agility, and business value goals.

# Root Cause
High pressure on C-suite and IT leaders to choose a solution for mission-critical SAP modernization

# Trigger Event
SAP modernization strategy is selected without proven approaches or workload-specific tailoring

# Impact
Modernization may not cut costs, increase agility, or deliver transformational business value

# Linked Risks
- R120
- R121
- R123

# Stakeholders

- C-suite leaders
- IT leaders
- SAP teams
- IBM Consulting
- AWS

# Contextual Constraints

- SAP workloads modernized on AWS


# Detection Source
SAP workload assessment / modernization strategy review

# Standardized Risk Entry
If a mission-critical SAP workload is modernized using a poorly fitting path or platform, expected cost, agility, and business value outcomes may not materialize.

# Key Quote for Prompting
> "specific SAP workload requirements"

# Logic Pattern
IF SAP workload requirements are specific AND the chosen modernization path is not tailored THEN business value realization is at risk.
