# Description
Attempting a full manual or broad rewrite of sprawling, undocumented legacy codebases stalls project progression.

# Root Cause
Verbose, unwieldy, tightly coupled code bases with outdated or non-existent documentation.

# Trigger Event
Initializing an all-at-once broad modernization over major business functions.

# Impact
Project stagnation, exhausted budgets, high risk of complete modernization abandonment.

# Linked Risks

# Stakeholders

- Project Managers
- Executives

# Contextual Constraints

- Occurs when full-scale code transformation is chosen over modular strategies.


# Detection Source
Project Plan

# Standardized Risk Entry
Launching massive, non-phased modernization efforts on unstructured codebases leads to project stagnation and high failure rates.

# Key Quote for Prompting
> "Modernization projects tend to be huge and daunting undertakings."

# Logic Pattern
IF [Codebase is monolithic and undocumented] AND [Modernization scope is broad/un-phased] THEN [Project Failure due to Massive Scope Explosion]
