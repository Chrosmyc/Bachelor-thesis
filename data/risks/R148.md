# Description
During modernization, older or obsolete technologies can reduce developer productivity and create continuity risk when knowledge, tools, or vendor support are limited.

# Root Cause
The code base contains old or obsolete technologies with limited market knowledge, tooling, or support.

# Trigger Event
Developers need to modernize, maintain, or migrate functionality implemented in obsolete technologies; technology support becomes unavailable.

# Impact
Developer productivity decreases; tooling and knowledge are limited; business continuity may be exposed to risk when technologies become unsupported.

# Stakeholders

- Developers; architects; operations; business continuity owners; modernization team

# Contextual Constraints

- Applies where technologies in the code base are old
- obsolete
- difficult to work with
- or unsupported.

Source Code

# Standardized Risk Entry
If modernization depends on obsolete technologies with limited knowledge, tooling, or support, development productivity and continuity of the modernization program may be threatened.
