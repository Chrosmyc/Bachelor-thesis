# Description
Selecting an unsuitable modernization strategy may lead to project failure.

# Root Cause
Misalignment between strategy and organizational goals

# Trigger Event
Incorrect modernization planning decisions

# Impact
Delays, high costs, failed modernization

# Linked Risks
- Cost Overrun
- Stakeholder Conflict

# Stakeholders

- Project Manager
- Business Leadership
- IT Team

# Contextual Constraints

- Large-scale modernization projects


# Detection Source
Project Planning Documentation

# Standardized Risk Entry
Inappropriate modernization strategy selection may significantly increase project delays, costs, and failure probability.

# Key Quote for Prompting
> "One right move, such as an accurate strategy selection, can translate into numerous negative impacts like delayed projects and high costs, ultimately leading to project failure."

# Logic Pattern
IF modernization strategy is misaligned with business goals THEN project failure risk increases
