# Description
Simultaneous migration activities may unintentionally evolve into a big-bang modernization.

# Root Cause
Excessive simultaneous migration scope.

# Trigger Event
Concurrent code and database modernization.

# Impact
Feasibility risks, uncontrolled complexity, delayed delivery.

# Linked Risks
- Schedule Overrun
- Development Failure

# Stakeholders

- Project Manager
- Software Architect

# Contextual Constraints

- Applies when data and code migration occur simultaneously.


# Detection Source
Project Plan

# Standardized Risk Entry
Simultaneous code and database modernization can unintentionally escalate into a big-bang migration effort.

# Key Quote for Prompting
> "This approach can easily degrade to a 'big bang' approach."

# Logic Pattern
IF code migration AND database migration are executed simultaneously THEN big-bang modernization risk increases.
