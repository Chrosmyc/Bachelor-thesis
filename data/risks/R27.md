# Description
Knowledge gained during early modernization phases may be lost if different personnel are involved in later increments.

# Root Cause
Insufficient knowledge transfer across modernization teams.

# Trigger Event
Team changes between modernization increments.

# Impact
Reduced understanding of legacy architecture and target architecture, reduced modernization efficiency.

# Linked Risks
- Architecture Misunderstanding

# Stakeholders

- Project Manager
- Software Architect

# Contextual Constraints

- Applies in multi-phase modernization projects.


# Detection Source
Project Plan

# Standardized Risk Entry
Modernization knowledge acquired during earlier increments may be lost when teams change between project phases.

# Key Quote for Prompting
> "These benefits are lost, to some degree, if different individuals are involved."

# Logic Pattern
IF modernization teams change between increments AND knowledge is not documented THEN modernization understanding decreases.
