# Description
Moving to a new system based on unproven technology may create risk in the modernized environment.

# Root Cause
Target system technology may not be proven.

# Trigger Event
Modernization selects a new system or technology that is really new.

# Impact
Target system instability or failure to deliver expected benefits.

# Linked Risks

# Stakeholders

- Architects
- managers
- business users

# Contextual Constraints



# Detection Source
Architecture Decision / Vendor Information

# Standardized Risk Entry
Moving to a new system based on unproven technology may create risk in the modernized environment.

# Key Quote for Prompting
> "Biggest risk is if you go to the new system that hasn't any proven technology or it is really new."

# Logic Pattern
IF the target system uses unproven technology AND modernization depends on it THEN post-modernization stability and benefit realization are at risk.
