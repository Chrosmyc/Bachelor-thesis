# Description
Modernization can require detailed planning, time, resources, and many manual tasks, creating a risk of slower execution and resource strain.

# Root Cause
Manual modernization tasks and process complexity

# Trigger Event
Executing modernization activities without automation or AI support

# Impact
Longer modernization process, higher resource consumption, and slower delivery

# Linked Risks

# Stakeholders

- Developers
- architects
- operations teams
- project managers

# Contextual Constraints

- Applies where tasks and processes are not automated


# Detection Source
Process analysis; project plan; task backlog

# Standardized Risk Entry
Modernization may slow down when teams rely heavily on manual tasks rather than automation or AI-supported processes.

# Key Quote for Prompting
> "Reducing or eliminating manual tasks"

# Logic Pattern
IF modernization contains many manual tasks AND automation is not used THEN the process may slow and consume more resources.
