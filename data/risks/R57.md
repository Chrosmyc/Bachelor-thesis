# Description
Modernization focused only on executable code may miss data and objects that are strategically more important.

# Root Cause
Legacy approaches often focus on code, while organizational data may be the strategic asset.

# Trigger Event
A migration plan targets code transformation while neglecting data and objects.

# Impact
Critical assets remain unmanaged; migration may fail business priorities.

# Key Quote for Prompting
> For many organizations, the data is the strategic asset, rather than the code.

# Logic Pattern
IF migration focuses only on code AND data/objects are strategic THEN modernization may miss business value.
