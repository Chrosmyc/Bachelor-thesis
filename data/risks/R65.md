# Description
Moving parts of the legacy system to microservices may introduce bugs during the migration.

# Root Cause
Transformation of implementation artifacts and movement of legacy parts into microservices.

# Trigger Event
Migrating implementation artifacts and deploying migrated microservices.

# Impact
Bugs in migrated microservices; bug reports are produced during verification and validation.

# Linked Risks
- R70

# Stakeholders

- Engineers responsible for migration; testers; users

# Contextual Constraints

- Applies after moving parts of the legacy system to microservices.


# Detection Source
Test Cases / Bug Reports

# Standardized Risk Entry
If implementation artifacts are transformed into microservices, bugs may be introduced during the migration and must be detected through verification and validation.

# Key Quote for Prompting
> "Such tests are intend to identity bugs introduced during the migration."

# Logic Pattern
IF legacy parts are migrated to microservices THEN bugs may be introduced during migration.
