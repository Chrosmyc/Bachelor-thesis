# Description
Legacy systems may not be compatible with cloud APIs, frameworks, or cloud database paradigms.

# Root Cause
Mismatch between legacy technologies and cloud technologies

# Trigger Event
Migration to cloud services or cloud databases

# Impact
Refactoring effort, migration delays, increased costs

# Linked Risks
- Migration failure
- cost overrun

# Stakeholders

- Software Architect
- Developer

# Contextual Constraints

- Especially relevant for PaaS and NoSQL migration


# Detection Source
Source Code

# Standardized Risk Entry
Cloud migration may introduce incompatibilities between legacy systems and cloud technologies.

# Key Quote for Prompting
> “… specific requirements may be posed on the technical implementation”

# Logic Pattern
IF legacy interfaces differ from cloud interfaces THEN incompatibility risk occurs
