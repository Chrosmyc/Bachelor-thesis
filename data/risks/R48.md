# Description
Existing anomalies, structural inconsistencies, or incomplete data can propagate into the modernized environment if data quality management is insufficient.

# Root Cause
Data anomalies and structural inconsistencies exist in source data before migration.

# Trigger Event
Data is standardized or migrated without comprehensive quality assessment and improvement.

# Impact
Incomplete or inconsistent data in the modern system, reduced migration success, and operational defects.

# Linked Risks
- R58; R59

# Stakeholders

- Data quality teams; migration teams; business owners

# Contextual Constraints

- Banking-sector data migration initiatives.


# Detection Source
Operational Data

# Standardized Risk Entry
Data quality risk arises when source anomalies and inconsistencies are not assessed and remediated before migration.

# Key Quote for Prompting
> data anomalies affecting 13% of records

# Logic Pattern
IF source data has anomalies AND quality management is weak THEN defects propagate into the modernized system
