# Description
Capability gaps across business, process, people, platform, operations, or security may remain hidden before migration, weakening readiness and planning.

# Root Cause
Capability gaps across AWS Cloud Adoption Framework dimensions

# Trigger Event
Migration planning proceeds without a migration readiness assessment

# Impact
Required migration capabilities may be missed; TCO planning may be incomplete

# Linked Risks
- R116
- R117
- R118

# Stakeholders

- IT leaders
- in-house IT teams
- business stakeholders
- AWS Partner / IBM Consulting

# Contextual Constraints

- AWS modernization programs using readiness and assessment phases


# Detection Source
Migration readiness assessment

# Standardized Risk Entry
If modernization readiness is not assessed, cross-dimensional capability gaps may remain undetected and weaken the migration plan.

# Key Quote for Prompting
> "identifies gaps"

# Logic Pattern
IF readiness assessment is skipped THEN capability gaps may remain undetected.
