# Description
Capability gaps identified during assessment may not be fixed before migration execution, leaving the organization without a sufficient operational foundation.

# Root Cause
Missing or insufficient mobilize phase after readiness assessment

# Trigger Event
Migration plan is executed before assessed gaps are remediated

# Impact
Weaker operational foundation; slower or less clear migration decisions; less smooth cloud transition

# Linked Risks
- R114
- R115
- R117

# Stakeholders

- In-house IT teams
- IBM Consulting
- migration decision makers

# Contextual Constraints

- AWS MAP-style migration with assess
- mobilize
- migrate-and-modernize phases


# Detection Source
Mobilize phase / assessment outputs

# Standardized Risk Entry
If assessed capability gaps are not fixed during mobilization, the migration may proceed with an inadequate operational foundation.

# Key Quote for Prompting
> "fixing the capability gaps"

# Logic Pattern
IF capability gaps are known AND mobilization does not fix them THEN migration execution risk increases.
