# Description
Modernization projects frequently face strong time constraints caused by limited resources, changing requirements, and scarcity of experts.

# Root Cause
Scarcity of experts, lack of resources, changing requests during long-term projects

# Trigger Event
Execution of long-running modernization projects

# Impact
Delays, project pressure, reduced modernization quality

# Linked Risks
- Resource shortage; changing requirements

# Stakeholders

- Project Manager
- IT Decision-Maker

# Contextual Constraints

- Long-term modernization projects


# Detection Source
Project Plan

# Standardized Risk Entry
Time pressure in modernization projects can reduce project quality and increase delivery risk, especially when resources and expertise are limited.

# Key Quote for Prompting
> "time constraints to finish the modernization are the most prominent challenge"

# Logic Pattern
IF modernization projects are long-running AND resources are limited THEN time constraint risk increases.
