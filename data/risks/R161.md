# Description
If modernization is postponed for too long, later upgrade work can become more complex and costly.

# Root Cause
Delayed modernization decisions; accumulated technical debt before the modernization starts.

# Trigger Event
The organization waits too long before initiating modernization.

# Impact
Future upgrades become more complex and costly.

# Linked Risks

# Stakeholders

- CIO; IT leadership; business leadership; finance leadership

# Contextual Constraints

- Applies where modernization has been delayed and technical debt continues to accumulate.


# Detection Source


# Standardized Risk Entry
Delaying IT modernization can increase the complexity and cost of later upgrade initiatives because technical debt continues to accumulate before transformation begins.

> 

# Logic Pattern
IF modernization is delayed AND technical debt accumulates THEN future modernization becomes more complex and costly.
