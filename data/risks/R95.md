# Description
A modernization effort may be reversed after implementation because the chosen architecture does not satisfy organizational needs.

# Root Cause
Adoption of architectures without sufficient evaluation.

# Trigger Event
Migration to a new architecture.

# Impact
Reverse migration, wasted effort, additional costs.

# Linked Risks
- R98

# Stakeholders

- Architects
- Development Teams

# Contextual Constraints

- Architectural migration initiatives.


# Detection Source


# Standardized Risk Entry
Migration to a new architecture may later be reversed if the selected architecture fails to meet organizational needs.

# Key Quote for Prompting
> "Systems transformed into microservice were migrated back to monolithic applications."

# Logic Pattern
IF a new architecture is adopted AND organizational needs are not met THEN migration reversal risk occurs
