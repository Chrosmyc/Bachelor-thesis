# Description
Modernization projects often fail because they are not systematically evaluated or planned, leading to unforeseen trade-offs and failures.

# Root Cause
Lack of understanding and preparedness from companies; rushing into projects without analyzing reasons.

# Trigger Event
Rushing into modernization without evaluating alternatives.

# Impact
Project failure, lost investment, business disruption.

# Linked Risks
- Cost Overrun
- Time Overrun

# Stakeholders

- Project Manager
- Software Architect

# Contextual Constraints

- Applies particularly to large-scale enterprise modernization


# Detection Source
Project Plan / Literature Review

# Standardized Risk Entry
Inadequate upfront planning and risk analysis frequently lead to the failure of modernization projects.

# Key Quote for Prompting
> "Some research suggests that most migration projects fail because of poor planning and risk analysis"

# Logic Pattern
IF No sufficient planning AND Modernization starts THEN Project Failure becomes very likely
