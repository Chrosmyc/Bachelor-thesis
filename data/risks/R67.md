# Description
ADM-based migration may miss useful services when only a limited set of patterns is used.

# Root Cause
Only four patterns were initially presented although more useful patterns exist.

# Trigger Event
Using ADM patterns for service discovery.

# Impact
Useful services may remain undiscovered.
