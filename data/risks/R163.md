# Description
Outdated or inefficient processes may be carried into the modernized environment, producing expensive but still ineffective processes.

# Root Cause
Existing practices are accepted as fixed; teams force old processes into new technology.

# Trigger Event
Modern technology is introduced without reexamining current practices and processes.

# Impact
Inefficient processes remain; modernization benefits are limited; costs increase without process improvement.

# Linked Risks

# Stakeholders

- Process owners; business units; IT leadership; end users

# Contextual Constraints

- Applies when modernization updates technology but leaves work habits and process structures unchanged.


# Detection Source
Process Documentation / Interview / Project Plan

# Standardized Risk Entry
Modernization can preserve or amplify inefficient work practices if old processes are simply moved into the new technology environment.

# Key Quote for Prompting
> 

# Logic Pattern
IF new technology is added AND old processes are not reexamined THEN inefficient processes persist in the modernized environment.
