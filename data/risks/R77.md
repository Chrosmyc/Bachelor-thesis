# Description
Migration can require major additional effort when services must comply with constraints imposed by the target SOA environment.

# Root Cause
The target SOA environment imposes constraints on services.

# Trigger Event
Implementing services under infrastructure, communication, security, data-storage, or service-model constraints.

# Impact
Cost and schedule increases; some candidate services may become unattractive for the pilot.

# Linked Risks
- R78

# Stakeholders

- Developers
- architects
- project managers

# Contextual Constraints

- Applies when target SOA constraints are strict or not yet fully accounted for.


# Detection Source
Project Plan / Architecture Diagram / Interview

# Standardized Risk Entry
Target-environment constraints can turn apparently feasible service migration into a high-effort modernization task.

# Key Quote for Prompting
> "Compliance with constraints requires major effort"

# Logic Pattern
IF target SOA constraints are significant AND services must be made compliant THEN migration effort and risk increase.
