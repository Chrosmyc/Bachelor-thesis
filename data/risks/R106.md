# Description
Digital touchpoints, online/mobile use, external users, and data aggregators can dramatically increase transaction volumes and compute demand beyond original system design.

# Root Cause
Legacy and modernized systems must support higher digital volumes, speed, and data access demand.

# Trigger Event
Modernization exposes capabilities through digital channels, APIs, mobile, or data layers.

# Impact
Strained systems, insufficient computing resources, poor user experience.

# Linked Risks

# Stakeholders

- Users; external entities; architecture staff; IT operations; business owners

# Contextual Constraints

- Online/mobile computing
- seasonal spikes
- data aggregators
- API access.


# Detection Source
Operational Data

# Standardized Risk Entry
Digitization can create scalability risk when new digital layers and external access increase transaction and data demands.

# Key Quote for Prompting
> "process dramatically increased transaction volumes and operate at speeds greater than the systems were designed to handle"

# Logic Pattern
IF modernization adds digital channels AND demand increases sharply THEN scalability and performance strain may occur.
