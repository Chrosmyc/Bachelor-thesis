# Description
A modernized information system may only partially achieve the expected outcome when enterprise transformation and quality concerns are ignored.

# Root Cause
Transformation to enterprise solution and quality concerns are overlooked.

# Trigger Event
Modernization focuses on technical change while ignoring enterprise-level expectations and quality.

# Impact
Modernized system only partially satisfies stakeholder expectations.

# Linked Risks
- R49
- R51
- R52

# Stakeholders

- Higher management; users; customers; modernization team

# Contextual Constraints

- Applies when modernization is expected to produce an enterprise solution.


# Detection Source
Project Plan; Requirements documentation

# Standardized Risk Entry
If modernization ignores enterprise transformation and quality concerns, the resulting system may only partially meet the expected outcome.

# Key Quote for Prompting
> "This overlooking leads to a modernized information system that partially achieves the expected outcome."

# Logic Pattern
IF enterprise transformation AND quality concerns are ignored THEN expected outcomes may be only partially achieved.
