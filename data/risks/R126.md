# Description
Teams move into broader or more complex modernization without first building experience in Azure operations, governance, and cost management. This can weaken execution readiness for later modernization efforts.

# Root Cause
Inexperience with Azure and lack of foundational cloud operations exposure.

# Trigger Event
The organization attempts complex modernization before gaining cloud operations experience.

# Impact
Reduced readiness for broader cloud adoption, governance issues, cost management difficulty, and execution risk.

# Linked Risks
- R127
- R131
- R134

# Stakeholders

- Execution teams
- operations teams
- cloud strategy team
- technical leads.

# Contextual Constraints

- Applies when teams are inexperienced with Azure and need a phased path to cloud adoption.


# Detection Source
Interview / Project Plan / operational readiness assessment

# Standardized Risk Entry
Attempting complex modernization before building Azure operations capability can reduce execution readiness and increase adoption risk.

# Key Quote for Prompting
> [Paraphrased evidence from G10] Rehosting can build experience with Azure operations, governance, and cost management.

# Logic Pattern
IF teams lack Azure operations experience AND complex modernization starts THEN execution readiness may be insufficient.
