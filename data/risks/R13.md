# Description
Engineers may lose familiarity with the inner workings of the system due to automated processes.

# Root Cause
High reliance on automation tools (ACMTs).

# Trigger Event
Implementation of automated migration tools.

# Impact
Loss of system knowledge, inability to troubleshoot when automation fails.

# Linked Risks

# Stakeholders

- DevOps Engineer
- Software Architect

Interview Transcripts

# Standardized Risk Entry
Heavy reliance on automated modernization tools risks the loss of critical engineering knowledge regarding system internals.
