# Description
Non-parallel deployment strategies provide no backup system if failures occur.

# Root Cause
Legacy system is immediately replaced.

# Trigger Event
Deployment of modernization increment as operational system.

# Impact
Operational outage without recovery path.

# Linked Risks
- Downtime

# Stakeholders

- DevOps Engineer
- Project Manager

# Contextual Constraints

- Applies to direct replacement deployment strategies.


# Detection Source
Deployment Plan

# Standardized Risk Entry
Direct operational deployment without fallback systems increases outage risk if failures occur.

# Key Quote for Prompting
> "There is no fallback mechanism in the event of a system failure."

# Logic Pattern
IF operational deployment occurs without parallel fallback THEN outage risk increases.
