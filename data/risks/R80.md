# Description
A short demonstration schedule can make high-risk service features unjustifiable within the pilot.

# Root Cause
The organization has a short schedule for completing the initial demonstration.

# Trigger Event
Including high-risk services or features in a short-term feasibility demonstration.

# Impact
Schedule overrun, failed demonstration, or insufficient time to resolve technical uncertainty.

# Linked Risks
- R79

# Stakeholders

- Project managers
- developers
- service provider
- service consumers

# Contextual Constraints

- Applies to pilot migrations with fixed short timelines.


# Detection Source
Project Plan / Interview

# Standardized Risk Entry
When a migration pilot has a short schedule, high-risk service features may threaten successful delivery.

# Key Quote for Prompting
> "the organization had a short schedule on which to complete an initial demonstration"

# Logic Pattern
IF pilot schedule is short AND high-risk items are included THEN delivery risk becomes unjustifiable.
