# Description
A problematic workload is rehosted without modernization, so existing performance, reliability, architectural issues, and technical debt continue into the cloud environment. This can force later rework.

# Root Cause
Like-for-like migration is applied to a workload whose root causes require modernization.

# Trigger Event
A problematic workload is moved to Azure with minimal changes.

# Impact
Technical debt is carried forward, existing issues persist, and later rework may be required.
