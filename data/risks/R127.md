# Description
A workload that requires compatibility updates, such as updates related to legacy technologies, unsupported SDKs, or SaaS adoption, is not modernized when needed. This can prevent the workload from supporting the selected migration path.

# Root Cause
Compatibility gaps are not identified or not justified with a business case.

# Trigger Event
Migration proceeds without modernization work needed for compatibility.

# Impact
Compatibility gaps persist, target strategy may not work as intended, and later corrective modernization may be needed.

# Logic Pattern
IF compatibility updates are required AND modernization is skipped THEN the migration path may not support the target state.
