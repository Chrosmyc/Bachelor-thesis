# Description
Modernization may encounter integration or migration problems if the system integration and migration strategy is not developed early.

# Root Cause
Integration and migration strategy is deferred or not documented.

# Trigger Event
Transformation begins before dependencies, interfaces, and migration strategy are clarified.

# Impact
Integration delays, migration issues, rework, and increased uncertainty during transformation.
