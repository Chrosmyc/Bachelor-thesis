# Description
If integration is not prioritized, modernization may stack silos higher instead of reducing them.

# Root Cause
CIOs assume system upgrades automatically improve data integrity; integration is not prioritized.

# Trigger Event
Systems are upgraded or migrated without unifying data across vendors and platforms.

# Impact
Increased complexity; persistent or deeper silos; lower usability of modernized systems.

# Linked Risks

# Stakeholders

- CIO; data teams; platform teams; vendors; business users

# Contextual Constraints

- Applies in multi-vendor and multi-platform modernization environments.


# Detection Source
Architecture Diagram / Data Lineage / Data Governance Documents

# Standardized Risk Entry
Modernization can amplify data and integration silos when system upgrades proceed without a deliberate cross-platform data strategy.

> 

# Logic Pattern
IF systems are upgraded AND integration is not prioritized THEN modernization can deepen data silos.
