# Description
Initial cost estimations for migration (finance, time, personnel) are significantly lower than actual requirements.

# Root Cause
Underestimating complexity and hidden costs (e.g., service wrapping vs. subsequent replacement).

# Trigger Event
Lack of insight into required migration budget and legacy system complexity.

# Impact
Financial waste; project delays; potential project cancellation. In one case, costs were 500% over budget.

# Linked Risks
- Resource Shortage

# Stakeholders

- Project Manager
- Financial Department

# Contextual Constraints

- Especially high risk when "service wrapping" hides the need for future deep code replacement.


# Detection Source
Initial Budget vs. Actual Spend

# Standardized Risk Entry
Underestimating modernization costs due to legacy complexity can lead to massive budget overruns and wasted resources.

# Key Quote for Prompting
> "The real cost and time were around 500% more than the initial estimated budget."

# Logic Pattern
IF [Lack of insight on required budget] AND [Migration starts] THEN [Waste of money/Overrun occurs]
