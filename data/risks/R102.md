# Description
Deployment may rely on external third parties, introducing new relationships, additional complexity, and potential risks.

# Root Cause
External coordination and reliance on partners or vendors during transition.

# Trigger Event
Deployment requires external third-party participation or service provider coordination.

# Impact
Coordination failures, delays, integration problems, increased complexity.

# Linked Risks
- R101
- R99

# Stakeholders

- Vendors; commercial service providers; IT; contracting; program manager

# Contextual Constraints

- Cloud migration
- vendor-supported deployments
- outsourced services.


# Detection Source
Project Plan

# Standardized Risk Entry
Deployment risk increases when modernization relies on third parties whose activities must be coordinated with internal workstreams.

# Key Quote for Prompting
> "coordination with and reliance on external third parties"

# Logic Pattern
IF deployment depends on external third parties THEN coordination complexity may create additional transition risk.
