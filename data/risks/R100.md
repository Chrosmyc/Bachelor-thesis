# Description
Roadmap estimates may be unreliable if they are guesses rather than based on a genuine assessment of the existing application.

# Root Cause
Insufficient current-state assessment and baseline understanding.

# Trigger Event
Cost and timeline planning is performed without detailed assessment.

# Impact
Underfunding, schedule slippage, weak business case, poor investment decisions.

# Linked Risks
- R105

# Stakeholders

- Program manager; finance; senior leadership; OMB/budget stakeholders

# Contextual Constraints

- Multiyear modernization initiatives requiring investment approval.


# Detection Source
Project Plan

# Standardized Risk Entry
Modernization estimates may be inaccurate when the roadmap is not grounded in a real assessment of the application and environment.

# Key Quote for Prompting
> "cost and time estimates are based on a genuine assessment of the existing application and – simply a guess"

# Logic Pattern
IF modernization estimates are not based on genuine assessment THEN cost and schedule assumptions may fail.
