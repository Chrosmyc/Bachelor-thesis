# Description
Legacy infrastructures may expose the organization to cyberattacks during modernization.

# Root Cause
Outdated security controls and unsupported dependencies

# Trigger Event
Exposure to modern cybersecurity threats

# Impact
Data breaches, ransomware attacks, compliance violations

# Linked Risks
- Reputational damage
- financial penalties

# Stakeholders

- Security Team
- Compliance Officers

# Contextual Constraints

- Systems lacking encryption or MFA


# Detection Source
Security Audit

# Standardized Risk Entry
Legacy modernization can expose organizations to cybersecurity incidents due to outdated protection mechanisms.

# Key Quote for Prompting
> "These systems often lack support for encryption, multifactor authentication, and real-time threat detection."

# Logic Pattern
IF outdated systems remain exposed during modernization THEN cybersecurity risk increases
